# Manikandan Muthiah — Portfolio

> **Live Site:** [pcbzmani.github.io/portfolio](https://pcbzmani.github.io/portfolio)

A personal portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, just clean and fast.

---

## About

This portfolio showcases my work as a **Senior Cloud Data Engineer** with 9 years of experience in Azure, GCP, Apache Spark, and Databricks. It includes my professional experience, certifications, side products, articles, and GitHub projects.

---

## Features

- **Animated particle background** — canvas-based floating particle network
- **Glassmorphism UI** — frosted glass cards with hover glow effects
- **Scroll reveal animations** — cards animate in as you scroll
- **Products section** — showcases MyFinance, Brain Games, SplitIt, and Live Translate
- **Dynamic copyright year** — always shows the current year
- **Fully responsive** — works on mobile, tablet, and desktop
- **PWA-ready links** — direct links to live apps and GitHub repos
